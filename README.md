# Mistral Reddit Weekly Sentiment (uv + Claude)

This app fetches the last week of posts and comments from `r/MistralAI`, sends the dataset to Claude for sentiment analysis, and returns JSON output containing:

- Average sentiment score (`1` to `5`)
- Weekly summary (generated by Claude)
- Top 3 posts by upvotes (link, content, upvotes)
- Keyword mention counts and links to mentions

## Requirements

- Python 3.11+
- [uv](https://docs.astral.sh/uv/)
- Anthropic API key

## Setup

1. Copy env template:

```bash
cp .env.example .env
```

2. Fill in values in `.env`:

- `REDDIT_USER_AGENT` (recommended)
- `ANTHROPIC_API_KEY`

3. Install dependencies:

```bash
uv sync
```

## Run

```bash
uv run mistral-sentiment --days 7 --subreddit MistralAI --output output/weekly_sentiment.json
```

If `--output` is omitted, JSON is still printed to stdout.

## Keywords

Edit `keywords.txt` to add or remove tracked terms. Use one term per line.

## Output shape

```json
{
  "subreddit": "MistralAI",
  "window": {
    "days": 7,
    "start_utc": "...",
    "end_utc": "..."
  },
  "counts": {
    "posts": 0,
    "comments": 0
  },
  "average_sentiment": 0,
  "summary_of_week": "...",
  "sentiment_method_notes": "...",
  "top_3_posts_by_upvotes": [
    {
      "post_link": "https://www.reddit.com/...",
      "content": "...",
      "upvotes": 123
    }
  ],
  "keyword_mentions": {
    "Le Chat": {
      "count": 4,
      "links": ["https://www.reddit.com/..."]
    }
  }
}
```

## Notes

- Reddit endpoints and rate limits can change over time.
- The app uses Reddit public JSON endpoints only (no Reddit app credentials required).
- Claude input payload size is bounded by `CLAUDE_MAX_ITEMS` and `CLAUDE_MAX_CHARS_PER_ITEM` (optional env vars).
