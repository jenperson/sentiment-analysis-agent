# Mistral Reddit Weekly Sentiment

This app fetches Reddit posts and comments from `r/MistralAI`, sends the dataset to an LLM for sentiment analysis, and returns JSON output containing:

- Average sentiment score (`1` to `5`)
- Weekly summary (generated by Claude)
- Top 3 posts by upvotes (link, content, upvotes)
- Keyword mention counts and links to mentions

## Requirements

- Python 3.11+
- [uv](https://docs.astral.sh/uv/)
- Mistral AI API key for the default analysis path

## Setup

1. Copy env template:

```bash
cp .env.example .env
```

2. Fill in values in `.env`:

- `REDDIT_USER_AGENT` (recommended)
- `MISTRAL_API_KEY`

Optional provider overrides:

- `LLM_PROVIDER=mistral` or `LLM_PROVIDER=claude`
- `LLM_MODEL=` to force a specific model name
- `ANTHROPIC_API_KEY` if using Claude

3. Install dependencies:

```bash
uv sync
```

## Run

```bash
uv run mistral-sentiment --days 7 --subreddit MistralAI --output output/weekly_sentiment.json
```

If `--output` is omitted, JSON is still printed to stdout.

Default analysis provider and model:

- Provider: `mistral`
- Model: `mistral-medium-2508` (Mistral Medium 3.1 served by Mistral AI)

To use Claude instead:

```bash
uv run mistral-sentiment --provider claude --model claude-sonnet-4-6 --subreddit MistralAI --output output/weekly_sentiment.json
```

Optional date range mode (instead of trailing `--days`):

```bash
uv run mistral-sentiment --subreddit MistralAI --start-days-ago 14 --end-days-ago 7 --output output/range_14_to_7.json
```

- `--start-days-ago` is the older boundary.
- `--end-days-ago` is the newer boundary (defaults to `0` when omitted).

## Keywords

Edit `keywords.txt` to add or remove tracked terms. Use one term per line.

## Output shape

```json
{
  "subreddit": "MistralAI",
  "window": {
    "days": 7,
    "start_utc": "...",
    "end_utc": "..."
  },
  "counts": {
    "posts": 0,
    "comments": 0
  },
  "analysis": {
    "provider": "mistral",
    "model": "mistral-medium-2508"
  },
  "average_sentiment": 0,
  "summary_of_week": "...",
  "sentiment_method_notes": "...",
  "top_3_posts_by_upvotes": [
    {
      "post_link": "https://www.reddit.com/...",
      "content": "...",
      "upvotes": 123
    }
  ],
  "keyword_mentions": {
    "Le Chat": {
      "count": 4,
      "links": ["https://www.reddit.com/..."]
    }
  }
}
```

## Notes

- Reddit endpoints and rate limits can change over time.
- The app uses Reddit public JSON endpoints only (no Reddit app credentials required).
- Mistral is the default sentiment provider.
- Claude remains available as an optional provider.
- LLM input payload size is bounded by `LLM_MAX_ITEMS` and `LLM_MAX_CHARS_PER_ITEM`.
